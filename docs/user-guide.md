## 快速开始

### 系统要求

- Node.js >= 18.0.0
- Python >= 3.8
- yt-dlp (最新版本)

### 启动步骤

1. **启动后端服务**
```bash
npm start
```
后端将运行在 http://localhost:3000

2. **启动前端界面**
```bash
cd src/frontend
npm run dev
```
前端将运行在 http://localhost:5173

3. **访问应用**
在浏览器中打开 http://localhost:5173

## YouTube集成工作流

### 步骤1: 输入YouTube URL

1. 打开应用后，默认进入 **YouTube** 页面
2. 在输入框中粘贴YouTube视频URL
3. 从下拉菜单选择字幕语言（支持英文、中文等）
4. 点击 **Download** 按钮

**支持的YouTube URL格式:**
- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/embed/VIDEO_ID`

### 步骤2: 下载字幕

下载完成后，系统会显示:
- 视频ID
- 视频标题
- 已下载的语言列表

### 步骤3: 处理和导入

1. 点击 **Process & Import** 按钮
2. 系统将:
   - 使用Python脚本清洗字幕数据
   - 移除时间戳和格式标记
   - 自动创建TXT资产
   - 保存视频元数据（ID、标题、语言）

导入成功后会显示:
- 资产ID
- 资产标题
- 语言
- 内容预览

### 步骤4: 风格转换

1. 点击 **Continue to Convert** 按钮
2. 在Convert页面:
   - 选择刚创建的资产
   - 选择写作风格
   - 点击 **开始转换**
3. 生成完成后，在Articles页面查看结果

## 手动资产管理

### 创建TXT资产

1. 点击导航栏的 **资产管理**
2. 在"创建新资产"卡片中:
   - 输入资产标题
   - 输入或粘贴文本内容
   - 点击 **创建资产** 按钮

### 查看资产列表

资产列表显示:
- 资产标题
- 内容预览（前200字符）
- 创建时间

### 删除资产

点击资产卡片右侧的 **删除** 按钮即可删除资产。

---

## 风格转换

### 转换资产为文章

1. 点击导航栏的 **风格转换**
2. 选择要转换的资产
3. 选择写作风格:
   - 选中风格的卡片会高亮显示
4. 点击 **开始转换** 按钮

### 查看转换结果

转换成功后会:
- 自动保存到文章列表
- 在页面下方显示转换结果
- 可通过Articles页面查看完整文章

### 选择风格说明

| 风格 | 描述 |
|------|------|
| professional | 专业商务风格 |
| casual | 轻松对话风格 |
| academic | 学术论文风格 |
| blog | 博客文章风格 |
| news | 新闻报道风格 |

## 文章管理

2. 点击文章卡片的 **展开** 按钮查看完整内容

## 常见问题

### Q: 下载失败怎么办？

**A:** 请检查:
1. YouTube URL是否正确
2. 视频是否有字幕可用
3. yt-dlp是否正确安装
4. 网络连接是否正常

### Q: 找不到处理后的文件？

**A:** 请确保:
1. 字幕下载成功
2. Python脚本路径正确
3. storage/processed 目录存在且有写入权限

### Q: 风格转换失败？

**A:** 请确认:
1. 已选择资产和风格
2. AI模型API配置正确
3. 资产内容不为空

### Q: 如何查看资产来源？

**A:** 从YouTube创建的资产会包含以下元数据:
- `source_video_id`: YouTube视频ID
- `source_video_title`: 原视频标题
- `source_language`: 字幕语言
- `source_type`: "youtube"

### Q: 前端页面空白？

**A:** 请检查:
1. 后端服务是否运行 (http://localhost:3000)
2. 浏览器控制台是否有错误
3. 运行 `npm run build` 验证构建
